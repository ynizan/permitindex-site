When using star cutout with borders, the star background color MUST match the border color.

/* Correct implementation */ .bordered-element { border: 2px solid var(--primary); position: relative; overflow: hidden; } .bordered-element::before { content: ''; position: absolute; top: -8px; right: 18px; width: 18px; height: 18px; background: var(--primary); /* MUST match border! */ clip-path: polygon(...); }

Why This Matters

  • Mismatched colors look unprofessional and confusing
  • Breaks visual hierarchy and consistency
  • Creates visual noise that distracts users
  • Violates design system principles
  • Makes the site appear poorly maintained
