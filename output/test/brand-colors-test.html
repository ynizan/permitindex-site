Usage Guidelines

‚úÖ DO

  • Use var(--primary) for all headers and links
  • Use var(--accent) for ONE primary CTA per page
  • Use var(--text) for body content
  • Use var(--text-light) for secondary info
  • Use var(--bg-light) for page backgrounds
  • Use var(--border) for all borders
  • Test contrast ratios for accessibility

‚ùå DON'T

  • Don't use hardcoded hex values (#003366)
  • Don't use Tailwind color classes (text-blue-600)
  • Don't overuse accent orange - it should be rare!
  • Don't use colors at low opacity (fails accessibility)
  • Don't create new colors without updating variables.css
  • Don't use accent orange for large text blocks

Code Examples

Correct Usage

<!-- Using CSS variables -->
<div style="background: var(--bg); color: var(--text);">
  <h2 style="color: var(--primary);">Heading</h2>
  <p style="color: var(--text-light);">Secondary text</p>
  <button style="background: var(--accent); color: white;">CTA</button>
</div>

Incorrect Usage

<!-- DON'T use hardcoded colors or Tailwind classes -->
<div class="bg-gray-50 text-gray-900">
  <h2 class="text-blue-600">Heading</h2>
  <p style="color: #666666;">Text</p>
  <button style="background: #FF6B35;">Button</button>
</div>
